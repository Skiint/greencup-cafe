# Starbucks Redesign Concept ☕

Современный концепт редизайна главной страницы Starbucks, разработанный с использованием **HTML, CSS и JavaScript** без использования фреймворков и backend.

## О проекте

Основная цель проекта — продемонстрировать навыки верстки, создания современного пользовательского интерфейса и реализации интерактивных элементов. Особое внимание уделено визуальному оформлению, адаптивности и пользовательскому опыту.

Проект включает:

- Адаптивную верстку;
- Современный UI-дизайн;
- CSS-анимации и плавные переходы;
- Интерактивные элементы на JavaScript;
- Структурированный и легко расширяемый код.

## Примечание

В проекте намеренно отсутствуют оригинальные изображения, логотипы и другие медиафайлы, защищенные авторским правом. Вместо них используются заглушки или предполагаются собственные изображения.

Основной акцент сделан **не на наполнении контентом**, а на разработке дизайна, структуры интерфейса и фронтенд-реализации.

## Технологии

- HTML5
- CSS3
- JavaScript (ES6)

## Статус проекта

🚧 Концепт фронтенд-редизайна. Backend и серверная логика отсутствуют.
